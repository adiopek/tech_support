# Instrukcja uruchomienia

## Wymagania
- Docker
- Docker Compose
- make

## Uruchomienie
1. Sklonowanie repozytorium.
2. `make init`.
3. Dokumentacja API (Swagger UI) będzie dostępna pod `http://localhost:8000/api`.
